Requires at least: 5.0

== Description ==

The Currency Rate Plugin displays the current exchange rates for specified currencies using a shortcode.

This plugin fetches exchange rates for USD and EUR relative to a base currency of your choice.

== Installation ==

1. Create a folder for you plugin here `/wp-content/plugins/[DIR_NAME]`
2. Upload the `currency-rate` folder to the `/wp-content/plugins/[DIR_NAME]` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Use the shortcode `[currency_rate currency_a="USD" currency_b="EUR"]` on any page or post where you want to display the exchange rate.
